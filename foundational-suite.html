The Quality Standards

Every document in the Foundational Suite must pass four rigorous tests before inclusion:

1. The Clarity Test

Can a Year 10 student understand the instructions?

2. The Completeness Test

Does it answer every question a user might have?

3. The Standalone Test

Can someone use it without asking for help?

4. The Consistency Test

Does it align with all other Suite documents?
